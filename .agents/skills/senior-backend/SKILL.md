Use this skill when designing, implementing, or reviewing a multitenant ecommerce backend system. 
This includes product catalog management, inventory control with stock movements, shopping cart, orders, user roles, and secure data isolation using shop_id.

The skill is especially relevant for systems built with Node.js, TypeScript, Supabase, and PostgreSQL, applying Row Level Security (RLS) for tenant isolation.

It should be used when working with:
- Multitenant architectures (SaaS)
- Ecommerce workflows (products, cart, orders, payments)
- Inventory systems with audit trails (inventory_movements)
- Backend API design and business logic
- Database modeling with relational structures and variants (size, color)
- Order lifecycle and status management
- Secure access control per tenant

The skill ensures scalable architecture, clean code practices, and strict data isolation between tenants.
